# 🦊 Firefox Profile Maker - Подробная инструкция для начинающих

## 🎯 Что это за программа?

**Firefox Profile Maker** - это специальная программа, которая помогает тебе создать безопасный профиль для браузера Firefox. 

### Простыми словами:
- **Браузер** - программа для просмотра интернет-сайтов (как Google Chrome, Safari)
- **Firefox** - один из таких браузеров, известный своей безопасностью
- **Профиль** - это набор настроек браузера (какие расширения установлены, какие настройки включены)
- **Эта программа** автоматически создает профиль с уже настроенной защитой от слежки в интернете

## 🛡️ Зачем это нужно?

Когда ты пользуешься интернетом, многие сайты:
- **Следят за тобой** - записывают, какие сайты ты посещаешь
- **Показывают навязчивую рекламу**
- **Собирают твои личные данные** для продажи рекламодателям
- **Замедляют загрузку сайтов** из-за трекеров

Эта программа защищает тебя от всего этого!

## 🔧 Что делает программа?

Программа автоматически устанавливает в Firefox полезные расширения:

### 🛡️ Расширения для защиты:
1. **uBlock Origin** - блокирует рекламу и трекеры
2. **Privacy Badger** - защищает от невидимого отслеживания
3. **HTTPS Everywhere** - делает соединения с сайтами безопасными
4. **CanvasBlocker** - защищает от продвинутых методов слежки
5. **ClearURLs** - очищает ссылки от кода для отслеживания
6. **Cookie AutoDelete** - автоматически удаляет cookies после закрытия вкладки
7. **Decentraleyes** - защищает от слежки через CDN
8. **Temporary Containers** - изолирует вкладки друг от друга
9. **Multi-Account Containers** - разделяет разные аккаунты

## 🟢 ПРОСТОЙ СПОСОБ (Рекомендуется)

### Шаг 1: Установи Firefox
1. Перейди на сайт [https://www.mozilla.org/firefox/](https://www.mozilla.org/firefox/)
2. Нажми "Скачать Firefox"
3. Установи браузер как обычную программу

### Шаг 2: Установи расширения вручную
Открой Firefox и установи эти расширения одно за другим:

1. **uBlock Origin**: https://addons.mozilla.org/firefox/addon/ublock-origin/
2. **Privacy Badger**: https://addons.mozilla.org/firefox/addon/privacy-badger17/
3. **HTTPS Everywhere**: https://addons.mozilla.org/firefox/addon/https-everywhere/
4. **CanvasBlocker**: https://addons.mozilla.org/firefox/addon/canvasblocker/
5. **ClearURLs**: https://addons.mozilla.org/firefox/addon/clearurls/
6. **Cookie AutoDelete**: https://addons.mozilla.org/firefox/addon/cookie-autodelete/
7. **Decentraleyes**: https://addons.mozilla.org/firefox/addon/decentraleyes/
8. **Temporary Containers**: https://addons.mozilla.org/firefox/addon/temporary-containers/
9. **Multi-Account Containers**: https://addons.mozilla.org/firefox/addon/multi-account-containers/

#### Как установить расширение:
1. Кликни по ссылке
2. Нажми синюю кнопку "Добавить в Firefox"
3. Подтверди установку
4. Повтори для всех расширений

### Шаг 3: Готово!
Теперь твой Firefox защищен! Ты заметишь:
- Меньше рекламы
- Быстрее загружаются сайты
- Больше приватности

## 🔴 ПРОДВИНУТЫЙ СПОСОБ (Для тех, кто хочет изучать)

⚠️ **Внимание**: Этот способ требует работы с командной строкой (терминалом)

### Что нужно знать:
- **Терминал/Командная строка** - окно, где ты пишешь команды текстом
- **Python** - язык программирования, на котором написана программа
- **Django** - система для создания веб-сайтов
- **Виртуальная среда** - изолированное место для программы

### Требования к системе:
- Linux или macOS
- Python 3.6 или новее
- Права администратора

### Проблемы в текущей настройке:
1. **Программа написана на старой версии Django** (3.1.14)
2. **Не совместима с новыми версиями Python** (3.13)
3. **Требует установки дополнительных пакетов системы**

### Для продвинутых пользователей:

#### Вариант 1: Обновление программы
1. Обнови Django в файле `requirements.txt` до новой версии
2. Исправь код для совместимости
3. Установи необходимые системные пакеты

#### Вариант 2: Использование Docker
1. Установи Docker на свой компьютер
2. Запусти команды:
   ```bash
   make build
   make up
   ```

#### Вариант 3: Ручная настройка
1. Создай виртуальную среду Python
2. Установи зависимости
3. Создай Django проект
4. Настрой конфигурацию

## ❓ Часто задаваемые вопросы

### В: Это безопасно?
О: Да! Программа только создает конфигурационные файлы. Все расширения - официальные и безопасные.

### В: Это бесплатно?
О: Да, полностью бесплатно и с открытым исходным кодом.

### В: Будет ли Firefox работать медленнее?
О: Наоборот! Блокировка рекламы и трекеров ускорит загрузку сайтов.

### В: Что если сайт перестанет работать?
О: Можно временно отключить uBlock Origin на конкретном сайте.

### В: Можно ли использовать с другими браузерами?
О: Эта программа работает только с Firefox, но многие расширения есть и для Chrome.
